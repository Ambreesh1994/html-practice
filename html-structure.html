HTML Page Structure

An HTML file is made up of elements enclosed in tags. These tags help define how your webpage looks and works.

Basic HTML Template:


<!DOCTYPE html>
<html>
  <head>
    <title>Page Title</title>
  </head>
  <body>
    <h1>This is a heading</h1>
    <p>This is a paragraph.</p>
  </body>
</html>

This basic structure is used in every webpage. Understanding this will help you write more organized and readable code.
