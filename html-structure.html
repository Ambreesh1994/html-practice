Basic HTML Page Structure

Every HTML document follows a common structure. Here's what it looks like:

    <!DOCTYPE html>
    <html>
      <head>
        <title>Page Title</title>
      </head>
      <body>
        <h1>Hello, world!</h1>
      </body>
    </html>

This format ensures the browser reads and displays your webpage correctly.
